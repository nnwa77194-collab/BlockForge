# BlockForge

———————————————————————————

BlockForge es una plataforma dedicada a la comunidad de Minecraft donde los creadores pueden compartir, descubrir y descargar texturas de alta calidad para Java y Bedrock.

———————————————————————————

BlockForge is a platform dedicated to the Minecraft community where creators can share, discover, and download high-quality textures for Java and Bedrock.
